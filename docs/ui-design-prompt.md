# UI Design Inspiration Prompt

Use this prompt in ChatGPT, Claude, or Midjourney to generate UI design inspiration for PlumberOS:

---

**PROMPT:**

```
Create a modern SaaS dashboard UI design for a plumbing business management platform called "PlumberOS".

**PRODUCT DESCRIPTION:**
PlumberOS is a all-in-one software platform for plumbing companies to manage their business. It includes:
- AI receptionist that answers calls 24/7 and schedules appointments
- Lead management and distribution system
- Job tracking for plumbers in the field
- Customer relationship management
- Invoicing and payment processing
- Analytics and reporting

**TARGET USERS:**
- Small to medium plumbing company owners (5-20 plumbers)
- Field plumbers who need to see their jobs
- Office staff who handle calls and scheduling

**DESIGN VIBE:**
- Professional, trustworthy, but modern
- Clean and minimal - not cluttered
- Inspired by top SaaS products like: Linear, Raycast, Vercel, HubSpot, Stripe Dashboard
- Enterprise-quality but approachable for non-technical users

**COLOR PALETTE:**
- Primary: Deep blue (#1e40af or similar)
- Secondary: Vibrant accent (maybe teal or orange for CTAs)
- Clean white backgrounds with subtle gray borders
- Dark text for readability

**KEY SCREENS TO DESIGN:**

1. **Main Dashboard** - Company overview with:
   - Stats cards (leads today, jobs in progress, revenue this month)
   - Recent leads list with status badges
   - Quick actions (add lead, add job)
   - Simple chart showing weekly jobs/revenue

2. **Leads Page** - List/table view with:
   - Lead source, customer name, issue, status, assigned plumber
   - Filters and search
   - Bulk actions

3. **Jobs Board** - Kanban or list view with:
   - Job cards showing plumber, customer, time, status
   - Drag-and-drop feel for status changes

4. **AI Receptionist Settings** - Configuration page with:
   - Phone number display
   - AI voice settings
   - Greeting customization
   - Call handling rules

**STYLE REQUIREMENTS:**
- Use plenty of white space
- Subtle shadows and rounded corners (8-16px radius)
- Clear typography (Inter, SF Pro, or similar sans-serif)
- Icon style: Lucide or similar minimalist icons
- Data tables should be clean with alternating row colors or borders
- Use badges/tags for status (green=completed, yellow=pending, red=urgent)

**INCLUDE IN THE DESIGN:**
- Sidebar navigation (collapsed and expanded states)
- Top bar with company name, user avatar, notifications
- Mobile-responsive layout hints
- Empty states for new users
- Loading states/skeletons

Generate a high-quality UI mockup in a sleek, modern SaaS aesthetic.
```
